## SETUP

1. Install node
2. Clone project
3. cd into server folder
4. Run npm install
5. Repeat for client folder

## SERVER RUN

-   Runs on port 5000
-   Use `npm run dev` to start project
-   If debug is needed:
    -   Toggle Auto Launch: Only With Flag in VSCode settings (Ctrl + Shift + P)
    -   Use `npm run debug` to start debug project
